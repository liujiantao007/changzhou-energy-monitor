# 依赖库说明

本项目需要以下依赖库，请下载并放置到本目录：

1. **ECharts**：用于图表和地图展示
   - 下载地址：https://cdn.jsdelivr.net/npm/echarts@5.4.3/dist/echarts.min.js
   - 文件名：echarts.min.js

2. **SheetJS**：用于解析Excel文件
   - 下载地址：https://cdn.jsdelivr.net/npm/xlsx@0.18.5/dist/xlsx.full.min.js
   - 文件名：xlsx.full.min.js

请确保这些文件已正确下载并放置在lib目录中，以确保项目在无网络环境下正常运行。
